# Ng7Advanced

1. 更新 Angular CLI 工具

    ```txt
    Mac: sudo npm i -g @angular/cli
    Win: npm i -g @angular/cli
    ```

2. 建立新專案

    ```sh
    ng new ng7-advanced --routing --style css
    code .
    npm start
    ```

3. Windows 開發人員需調整輸入法設定
